🔥 Free Fire MAX ID Value Website

📌 Project Overview

This is a gaming-themed website that calculates the approximate value of a Free Fire MAX account based on account items such as level, gun skins, elite passes, bundles, emotes, and diamonds spent.

The website also includes:

- Login system (local storage)
- ID value calculator
- Leaderboard system
- Animations and gaming UI
- Single file website (easy to host on GitHub Pages)

---

🎮 Features

- User Login (stored in browser)
- Enter Player Name and UID
- Calculate Estimated Account Value
- Save player data
- Leaderboard (Top account values)
- Animated gaming interface
- Works on mobile and desktop
- No database required (uses LocalStorage)

---

🧮 Value Calculation Formula

The estimated account value is calculated using this logic:

Item| Value
Account Level| Level × 10
Gun Skins| Gun Skins × 200
Elite Pass| Elite Pass × 100
Rare Bundles| Bundles × 150
Emotes| Emotes × 50
Diamonds Spent| Diamonds × 0.5

Total Value = Sum of all above

---

🌐 How to Host on GitHub Pages

1. Go to GitHub
2. Create a new repository
3. Click Add file → Create new file
4. Name the file index.html
5. Paste the website code
6. Click Commit
7. Go to Settings → Pages
8. Enable GitHub Pages
9. Your website will be live at:

https://yourusername.github.io/repository-name

💾 Data Storage

This website uses LocalStorage, which means:

- Data is saved in the user's browser
- No server or database required
- Leaderboard works on the same device/browser
