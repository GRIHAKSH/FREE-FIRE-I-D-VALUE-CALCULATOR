## Commit Information – Free Fire ID Value Website

### Commit Title
Add Free Fire ID Value Calculator Website with Login, Leaderboard, and Animations

### Commit Description
This commit includes the full Free Fire MAX ID Value Calculator website built using HTML, CSS, and JavaScript in a single file. The website allows users to log in, enter account details, calculate an estimated account value, and view a leaderboard of calculated values. The project uses LocalStorage to store user and player data directly in the browser without a backend database.

### Features Added in This Commit
- Single page website (index.html)
- Gaming-themed dark UI
- Animated glowing title
- Navigation system (Home, Login, Calculator, Leaderboard)
- Local login system
- Free Fire ID value calculator
- Leaderboard system with sorting
- Data stored using LocalStorage
- Button hover animations
- Section transition animations
- Mobile-friendly layout
